# Jumping Bunny Game

Jumping Bunny is a small 2D endless jump game built using Python and Pygame.  
The player controls a bunny that jumps across moving platforms in a continuously scrolling world.  
The goal is to survive as long as possible while jumping from platform to platform.

The game is currently basic and not final. More improvements, levels, mechanics, and visual elements will be added in future updates.

## Features
- Smooth player movement with double jump
- Randomly generated platforms
- Endless side-scrolling world
- Simple and clean code structure

## Requirements
- Python 3.x
- Pygame


## How to Run
1. Download or clone this repository.
2. Place your image assets inside the correct paths or update the paths in the script.
3. Run the game using your app of chooise.


## Status
This game is still under development and will receive future updates and improvements.
